# Donezo

> Stay organized. Finish fast.

## How to Run

The app must be served over HTTP — opening `index.html` directly as a file won't work.

```bash
npx serve .
```

Then open `http://localhost:3000` in your browser.

## How to Use

- Click **+ new task** under any column to add a task
- Click **edit** on a card to update it
- Click **delete** on a card to remove it
- **Drag and drop** cards between columns with your mouse or finger
- Press **?** in the bottom right to see keyboard shortcuts

## Tech Stack

- HTML, CSS, JavaScript — no libraries or frameworks
- localStorage for data persistence
